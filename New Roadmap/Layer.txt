1️⃣ CONTENT LAYER (Ngân hàng câu hỏi)

Đây là question bank – nơi giáo viên tạo nội dung.

question_categories
        │
question_types  (map → engine)
        │
      questions
      │      │
      │      ├──────── options
      │      │
      │      ├──────── question_text_answers
      │      │
      │      └──────── passages
      │                 │
      │                 └── passage_types
      │
      └──────── question_tags
                   │
                   └── tags
Vai trò từng bảng
Table	Vai trò
question_categories	Phonetics, Reading...
question_types	A1, B2, C1…
questions	câu hỏi
options	MCQ answers
question_text_answers	text answers
passages	reading/listening content
2️⃣ ENGINE LOGIC LAYER

Đây là cách engine sử dụng data.

ENGINE

MCQ_SINGLE
  → options

TEXT_INPUT
  → question_text_answers

PASSAGE_MCQ
  → passages + options

PASSAGE_TEXT
  → passages + question_text_answers

REORDER
  → question_text_answers

TRUE_FALSE
  → options

AUDIO_MCQ
  → passages(audio) + options

AUDIO_TEXT
  → passages(audio) + question_text_answers

ESSAY
  → questions only

3️⃣ Audit workflow cho mỗi table

Mỗi table ta check 5 thứ:

Step 1 — RLS enabled?
ALTER TABLE table_name ENABLE ROW LEVEL SECURITY
Step 2 — GRANT permission

Check:

anon
authenticated
service_role
Step 3 — policies list
SELECT *
FROM pg_policies
WHERE tablename = 'table_name';
Step 4 — policy logic check

Ví dụ:

check	example
student own row	user_id = auth.uid()
teacher own exam	created_by = auth.uid()
public read	USING (true)
Step 5 — attack vectors

Check xem policy có lỗi:

lỗi	ví dụ
missing WITH CHECK	insert bypass
USING true	data leak
no owner filter	teacher thấy exam người khác

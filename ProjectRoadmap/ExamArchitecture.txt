5️⃣ RPC #1 (quan trọng nhất)
generate_exam_from_template()

Đây là engine random câu hỏi.

Input:

template_id
student_id

Engine sẽ:

for each template_section
    for each template_item
        random questions
            category
            difficulty
            grade
            type

Insert:

exams
exam_questions
6️⃣ Example Flow

Template rule:

Grammar section
5 MC questions
grade = 9
difficulty 2-3
category = grammar

Engine query:

SELECT id
FROM questions
WHERE
    category_id = grammar
    AND difficulty BETWEEN 2 AND 3
    AND grade_level = 9
    AND is_active = true
ORDER BY random()
LIMIT 5

Insert:

exam_questions

Ví dụ:

id	section_id	category_id	question_type_id	quantity

   pronunciation : 5 câu
   stress pattern : 3 câu

Vocabulary
   synonym : 5 câu

4. Generate exam runtime

Khi student bấm Start Exam

RPC sẽ:

generate_exam_from_template(template_id, student_id)
Step 1 — tạo exam

Insert:

exams
id	template_id	user_id	status
100	1	student_1	in_progress
Step 2 — đọc template_items
SELECT *
FROM exam_template_items
WHERE template_id = 1
Step 3 — random questions

Ví dụ item:

phonetics
pronunciation
quantity = 5

query:

SELECT *
FROM questions
WHERE category_id = phonetics
AND question_type_id = pronunciation
ORDER BY random()
LIMIT 5
Step 4 — insert exam_questions

Ví dụ:

id	exam_id	question_id	order
1	100	52	1
2	100	12	2
3	100	88	3
5. Student làm bài

Student answer → lưu:

user_answers
id	exam_question_id	user_id	answer
6. Submit exam

Update:

exams.status = submitted
